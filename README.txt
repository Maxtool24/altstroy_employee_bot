Altstroy Employee Bot
Описание
Telegram-бот Altstroy Employee Bot предназначен для автоматизации процессов управления отчетами, сменами, инструментами, материалами и бригадами сотрудников строительной компании. Бот позволяет сотрудникам отправлять отчеты, управлять своими сменами, запрашивать материалы и просматривать информацию о бригадах и инструментах. Администраторы имеют доступ к просмотру всех отчетов и смен.

Основные функции
Регистрация сотрудников:

Сотрудники могут зарегистрироваться в системе с помощью команды /register.

Отправка отчетов:

Сотрудники могут отправлять текстовые отчеты с помощью команды /report.

Управление сменами:

Сотрудники могут начинать и завершать смены с помощью команд /shift_start и /shift_end.

Просмотр своих смен доступен через команду /view_shifts.

Управление бригадами:

Администраторы могут создавать бригады с помощью команды /create_brigade.

Просмотр списка бригад доступен через команду /view_brigades.

Управление инструментами:

Администраторы могут добавлять инструменты с помощью команды /add_tool.

Просмотр списка инструментов доступен через команду /view_tools.

Заявки на материалы:

Сотрудники могут запрашивать материалы с помощью команды /request_material.

Административные функции:

Администраторы могут просматривать все отчеты с помощью команды /view_reports.

Просмотр всех смен доступен через команду /view_all_shifts.

Примеры использования
1. Регистрация сотрудника
Пользователь отправляет команду /register:

Copy
/register
Бот отвечает:

Copy
Вы успешно зарегистрированы!
2. Отправка отчета
Пользователь отправляет команду /report с текстом отчета:

Copy
/report Сегодня выполнили укладку плитки в коридоре.
Бот отвечает:

Copy
Отчет успешно отправлен!
3. Начало смены
Пользователь отправляет команду /shift_start:

Copy
/shift_start
Бот отвечает:

Copy
Смена начата!
4. Завершение смены
Пользователь отправляет команду /shift_end:

Copy
/shift_end
Бот отвечает:

Copy
Смена завершена!
5. Просмотр своих смен
Пользователь отправляет команду /view_shifts:

Copy
/view_shifts
Бот отвечает:

Copy
Начало: 2023-10-01 08:00:00, Конец: 2023-10-01 17:00:00
Начало: 2023-10-02 08:00:00, Конец: 2023-10-02 17:00:00
6. Создание бригады
Администратор отправляет команду /create_brigade с названием бригады:

Copy
/create_brigade Бригада №1
Бот отвечает:

Copy
Бригада "Бригада №1" создана!
7. Просмотр бригад
Пользователь отправляет команду /view_brigades:

Copy
/view_brigades
Бот отвечает:

Copy
ID: 1, Название: Бригада №1
ID: 2, Название: Бригада №2
8. Добавление инструмента
Администратор отправляет команду /add_tool с названием и количеством инструмента:

Copy

Бот отвечает:

Copy
Инструмент "Молоток" добавлен!
9. Просмотр инструментов
Пользователь отправляет команду /view_tools:

Copy
/view_tools
Бот отвечает:

Copy

Администратор отправляет команду /view_reports:

Copy
/view_reports

Copy

Установка и запуск
1. Установка зависимостей
Установите необходимые зависимости с помощью команды:

bash
Copy
pip install -r requirements.txt
2. Настройка конфигурации
Откройте файл config.py и укажите токен бота и ID администратора:

python
Copy
TOKEN = "ВАШ_ТОКЕН"
ADMIN_ID = 123456789  # Замените на ID администратора
3. Запуск бота
Запустите бота с помощью команды:

bash
Copy
python run.py
